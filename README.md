# 🚀 AI TSP Optimizer

AI Optimization Simulator for solving the **Traveling Salesman Problem (TSP)** using Artificial Intelligence algorithms with realtime visualization and futuristic UI.

🌐 **Live Demo:**
https://sony-aitspoptimizer.my.id

---

# 📌 About Project

This project simulates how Artificial Intelligence algorithms optimize routes in the Traveling Salesman Problem (TSP).

The system visualizes route optimization in realtime using interactive canvas rendering and animated algorithm processing.

Built with modern frontend technologies and deployed online using Vercel with custom `.my.id` domain.

# 🧠 Algorithms Used

## 1. Hill Climbing

A local search algorithm that continuously improves the current solution by selecting better neighboring routes.

## 2. Simulated Annealing

A probabilistic optimization algorithm inspired by metal annealing processes to avoid local optimum traps.

## 3. Genetic Algorithm

An evolutionary optimization algorithm using selection, crossover, and mutation techniques inspired by natural genetics.

